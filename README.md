# aditya-negi

Personal portfolio — dark, techy, single-page. Full-stack developer & builder.

Built with plain **HTML + CSS + vanilla JS** (no framework, no build step). Deployed on Vercel.

## Sections
About · Stack · Work · Process · Resume · Contact

## Run locally
```bash
# just open it
open index.html
# or serve it
python3 -m http.server 8000   # → http://localhost:8000
```

## Stack
HTML5 · CSS3 · Vanilla JS · Space Grotesk / Inter / JetBrains Mono

## Structure
```
index.html    markup + sections
styles.css    theme, layout, responsive
script.js     typewriter terminal, scroll reveal, nav
```
